页面布局 时段统计报表
操作 响应
界面初始化 时间段 默认显示 7天前的时间 （到） 今日时间
7天: 起始日期= 今天 - 6, 截至日期 = 今天
3个月: 起始日期= 今天 - 89, 截至日期 = 今天
半年: 起始日期= 今天 - 182, 截至日期 = 今天
调用接口/{shopId}/inv/getShopSkuTags, 获取商品标签列
调用接口/{shop}/report/v2/salehour/period/report得到报表内容
默认显示的列: 零售时段/零售总额/客单价/交易笔数/零售毛利额/毛利率 不翻页

客单价 = saleAmount / saleCount
毛利率 = saleGrossAmount / saleAmount
点击日期输入框 调用日期选择控件
点击 7天/3个月/半年 按照规则,修改 起始日期, 结束日期 输入框中的值
点击"查询" 组织查询的各项参数
调用接口/{shop}/report/v2/salehour/period/report得到报表内容
点击"重置" 查询中的各项条件, 回复初始值
点击 标签 下拉显示所有的商品标签, 可以多选打勾. 如已经有被选择的标签, 则默认打勾
当商品标签一个都没有选择时, 标签控件框, 默认显示无
当商品标签已经选择一个或多个时, 标签控件框, 默认显示"已选择"
最多勾选3个标签, 超过了hdtip提示: 最多只能选择3个商品标签!
勾选 一个标签 组织查询的各项参数
组织query参数的tagNames
调用接口/{shop}/report/v2/salehour/period/report得到报表内容
增加两列 : {标签名称}零售额 / {标签名称}零售数
