页面布局 经营分析报表UX页面
整体操作 响应
界面初始化 进入页面分析报表页面经营分析报表接口路径/{shopUuid}/report/operationanalysis

时间段 默认显示 7天前的时间 （到） 今日时间
汇总图 x轴展示销售额 进货2个种类
              y轴展示出默认搜索条件的 销售额[#F56E6A] 进货额[#4FA8F9] ,柱状图宽度默认30px,柱状图顶显示数值
              y轴最大值是所给数据最大值加上1000（最大值小于1w） 10000 （最大值小于100w）, 最小值是0, y轴值有10个间隔
              鼠标悬浮柱状图显示提示2个种类的种类名称(ex:销售额 进货额) 销售值(ex:2500 300)

趋势图 图表时间段 非加盟店 加盟店(1.9.5版本)都展示并可以修改
              夜班时间控件晚班时间展示默认 23:00 至 07:00
              折线图转折点标点统一为圆点
              x轴展示具体时间,x轴展示的数据个数不会超过8个,x轴数据总个数不会超过30个,在1000分辨率下 x轴数据每隔3个展示在图表上
              y轴最大值是所给数据最大值加上50（最大值小于0.1w） 1000（最大值小于1w） 10000 （最大值小于100w）, 最小值是0, y轴值有5个间隔,y轴一共有3条 图表左侧2条为 数量(笔) 金额(元), 右边为利率(%) Y轴不显示坐标刻度
              y轴展示出默认搜索条件的 销售额[#F56E6A] 客单价[#009999] 销售笔数[#6EC71E] 夜班销售额[#FFCC00] 鼠标悬浮显示种类名称及其对应值(ex:毛利额 300)
              种类legend全部展示为圆点颜色默认为销售额[#F56E6A] 客单价[#009999] 销售笔数[#6EC71E] 夜班销售额[#FFCC00]库存品项数[#1B1B1B] 动销品项数[#FF66CC] 进货[#4FA8F9] 毛利额[#ff6600] 毛利率[#cc3300]
销售额表格 固定展示列'日期'、'销售额'、'毛利额'、'毛利率'、'客单价'、'夜班销售总额'、'动销品项数'、'进货'
日期这列数据会根据所选 按天或月份 或者 年份展示不同时间单位的值
改变搜索条件 时间段 初始日期可以选择 一个时间
              结束日期可以选择 一个时间
              2个日期时间段间隔进行判断 如果查询时间段小于40天，则按天(传DAY)，如果大于40天，则按自然月(传MONTH)，如果大于365天，则按自然年传(YEAR)
最近时间段标签 可选择 '7天' '3个月' '半年'其中一个, 3个选项:近7天= (今天-6天) 近3个月 = (今天 - 89 天) 近半年 = (今天 -182天) (时间标签只是方便选择时间段不进行查询数据)
点击搜索按钮 获取数据，调用接口经营分析报表接口路径 /{shopUuid}/report/operationanalysis
趋势图和销售额图随着搜索条件改变展示数值也变化
点击默认按钮 初试时间和结束时间变成7天前时间和今日 夜班时间变成初始23:00 结束 07:00

趋势图图表夜班时间段 目前1.95版本 是否是加盟会员不影响 夜班时间段选择
              夜班时间段2个时间选择范围是0:00 ~ 23:00 选择完之后,点击查询 趋势图会根据所选夜班时间展示数据

点击修改指标按钮 弹出框展示 '销售额'、'毛利额'、'毛利率'、'客单价'、'夜班销售总额'、'库存品项数'、'动销品项数'、'进货' 9个勾选项
勾选项勾选完成后 点击确定按钮 弹窗关闭 趋势图只展示所勾选的勾选项折线图

销售额表格 固定展示列'日期'、'销售额'、'毛利额'、'毛利率'、'客单价'、'夜班销售总额'、'动销品项数'、'进货'
销售额表格 固定展示列表格数据随着顶栏搜索条件改变而变化
页面整体样式 最小分辨率时页面宽度为1000px,小于1000px时不在进行响应式变化
页面内图表会根据 分辨率改变进行响应式的变化
