页面布局 盘点单上传文件
操作 响应
界面初始化 判断页面操作权限，权限ID：“库存”
点击“选择文件”按钮 打开windows弹出框，供用户选择：“*.xls; *.xlsx”文件
在windows框中点击确定后 页面获得文件名
调用上传文件接口/{shop}/checkBill/upload
获得接口返回的 获得商品的重复记录（可能有） + 本次上传操作的“唯一ID”
if 无商品的重复记录 then
　路径跳转 盘点单导入，带上参数“本次上传操作的唯一ID”， “文件名”
if 有商品的重复记录 then
　显示“商品重复记录”浮层。
在“商品重复记录”浮层
点击“继续操作”		 路径跳转 盘点单导入，带上参数“本次上传操作的唯一ID”， “文件名”
在“商品重复记录”浮层
点击“取消”		 浮层关闭
