页面布局 库存调整记录查询
操作 响应
界面初始化 判断页面操作权限，权限ID：“库存”
界面显示的操作权限判断：无
获取“商品标签”数据，调用接口“商品标签
获取“库存调整记录查询”数据，调用接口“库存调整记录查询
默认带时间范围条件：本周

默认排序：调整时间逆序
补充：查询、清空按钮
其他：
本周：开始时间变为当前所在周的周一；结束日期为空；
本月：开始日期变为当前所在月份的第一天（1号）；结束日期为空；
近3个月：开始日期变为当前月份减2个月的第一天（例如现在是5月10日，开始日期变为3月1日），结束日期为空；
点击查询 合成条件，调用接口“库存调整记录查询
点击清空 将所有查询条件置为初始状态
合成条件，调用接口“库存调整记录查询
本月 获取“库存调整记录查询”数据，调用接口接口
默认带时间范围条件：本月
近3个月 获取“库存调整记录查询”数据，调用接口接口
默认带时间范围条件：近3个月
点击“名称规格” 可以跳转到《商品流水报表》，同时带过去的查询条件有：
调整时间所在月份（按日期：所在月的第一天——所在月的最后一天）； 业务类型上选定:[√]库存调整。
