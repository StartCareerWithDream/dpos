状态为: 空, 操作按钮: 缴款, 理论上, 应该不会出现, 因为进不来
状态为: 待签字, 操作按钮: 签字 打印 编辑 删除
状态为: 待处理, 操作按钮: 处理 作废
状态为: 已处理, 操作按钮: 无
