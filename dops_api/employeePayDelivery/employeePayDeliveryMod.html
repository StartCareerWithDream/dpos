操作 响应
界面初始化 判断页面操作权限，权限ID：缴款登记
界面显示的操作权限判断：[所有操作按钮: 暂时不判断权限 ]
获取 收银缴款单信息, 调用接口收银缴款单待定

备注可填写: [0, 100]
现金.实付款: 必填 maxlength=8, >=0, 金额保留两位2小数, 进入界面时默认选中
其他支付.实付款: 必填 maxlength=8, >=0, 金额保留两位2小数
改写"现金.实付款" 该行的"长短款"同步修改, 最下面的合计信息同步修改
点击"确认缴款"按钮 数据判断: 现金.实付款, 其他支付.实付款 必须填写
调用接口保存待定
if 失败 then hdtip('保存失败', 'error', 失败原因)
if 成功 then hdtip('保存成功', 'success'), 路径跳转 缴款单明细
