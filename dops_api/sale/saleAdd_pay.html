页面布局 销售收银及结算
操作 响应
初始化 如有会员数据对象, 则显示会员信息. 会员输入框的操作与顶栏相同, 不再复述
合计/现金 默认显示"商品数据对象"的金额合计
应收金额 = 支付数据对象中的应收金额, 默认为"合计"金额
根据"默认支付方式", XX支付方式样式选中

屏幕快捷键列表:
1. F1: 如果尚未开通"在线支付", 显示开通, 进入lion支付方式设置; 否则"扫描支付"
2. F2: 更多支付方式的对话框
3. F3: 现金支付选中
2. F4: 显示折扣弹出框
3. F5: 储值支付弹出框, 调用接口/{shop}/member/get, 得到最新会员储值余额信息.
4. F6: 会员, 会员输入框获得焦点
5. F7: 优惠券弹出框, 输入后, 点击确认，调用/{shop}/mp/calFav接口
5. F8: 当设置了抹零规则后, 对于"应收"按规则抹零
点击结算按钮 调用接口/{shop}/sale/v2/save
点击"确认", 关闭付款对话框
销售单所有数据对象清空
初始化销售, 获取门店配置中销售相关的内容,调用接口 /{shop}/sale/v2/create
调用收藏夹初始化接口/{shop}/skuFavorite/get/byShop
点击"确定并打印小票" 调用打印功能
关闭付款对话框.

初始化销售, 获取门店配置中销售相关的内容,调用接口 /{shop}/sale/v2/create
